Sorting Techniques in Python

Sorting is a common operation in programming used to arrange data in a specific order, such as ascending or descending. Python provides powerful built-in tools that make sorting simple and efficient. Whether you’re working with numbers, strings, or complex data structures, Python offers techniques that can handle sorting with minimal code.

The two primary tools for sorting in Python are the sorted() function and the list.sort() method. Both can sort data, but they differ slightly. The sorted() function returns a new sorted list, leaving the original data unchanged:
numbers = [5, 2, 9, 1]
sorted_numbers = sorted(numbers) # [1, 2, 5, 9]

On the other hand, list.sort() sorts the list in place and does not return a new list:
numbers.sort()
print(numbers) # [1, 2, 5, 9]

By default, sorting is done in ascending order. To sort in descending order, pass the reverse=True argument:
sorted_desc = sorted(numbers, reverse=True) # [9, 5, 2, 1]

Python allows sorting using custom criteria by providing a key argument. For example, to sort strings by their length:
words = ["apple", "banana", "fig", "cherry"]
sorted_words = sorted(words, key=len) # ['fig', 'apple', 'banana', 'cherry']

You can also sort complex data like lists of tuples or dictionaries. For instance, sorting a list of tuples by the second element:
pairs = [(1, 3), (2, 2), (3, 1)]
sorted_pairs = sorted(pairs, key=lambda x: x[1]) # [(3, 1), (2, 2), (1, 3)]

While Python’s built-in sorting is efficient and recommended for most use cases, you can also implement classic algorithms like bubble sort, insertion sort, or quicksort for learning purposes. For example, a simple bubble sort repeatedly swaps adjacent elements if they are in the wrong order:
for i in range(len(numbers)):
  for j in range(len(numbers) - i - 1):
    if numbers[j] > numbers[j + 1]:
      numbers[j], numbers[j + 1] = numbers[j + 1], numbers[j]

In summary, Python offers simple yet powerful tools for sorting. The sorted() function and list.sort() method, combined with custom key functions and options like reverse, make sorting flexible and efficient for a wide range of data types and structures.
