Strings in Python

In Python, a string is a sequence of characters enclosed within quotes. Strings are one of the most commonly used data types in Python, used to represent textual data. You can create a string by enclosing characters in single quotes ('), double quotes ("), or triple quotes (''' or """) for multi-line strings. For example:
name = "Python"
greeting = 'Hello, World!'
message = '''This is a multi-line string.'''

Strings in Python are immutable. This means that once a string is created, its content cannot be modified. Any operation that appears to modify a string actually creates a new string. For example:
text = "Hello"
text = text + " World" # This creates a new string "Hello World"

You can access individual characters in a string using indexing, where the first character is at position 0:
letter = name[0] # This will be 'P'

Slicing allows you to extract a part of a string:
sub = name[1:4] # This will be 'yth'

Python provides many useful string methods. Some examples include:
name.lower() # Converts to lowercase
name.upper() # Converts to uppercase
name.replace("Py", "My") # Replaces substring
name.startswith("Py") # Checks if the string starts with "Py"
name.endswith("on") # Checks if the string ends with "on"

Strings can contain escape sequences for special characters:
newline = "First line\\nSecond line" # Inserts a newline
tabbed = "Column1\\tColumn2" # Inserts a tab space

You can iterate over a string with a loop:
for char in name:
    print(char)

In summary, strings in Python are powerful and versatile. They offer a rich set of methods for processing and manipulating text, and their immutability ensures consistency. Mastering strings is essential for working with input, output, files, and data in Python.
